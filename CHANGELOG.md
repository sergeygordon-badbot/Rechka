# История изменений

## 0.3.2 — 2026-07-16

- новое название «Речка» и цельная визуальная айдентика;
- новый контрастный значок для ярлыка, окна и области уведомлений;
- установщик удаляет старый ярлык «Голосовой ввод» при обновлении.

## 0.3.1 — 2026-07-16

- современный светлый интерфейс и новый значок приложения;
- компактная живая плашка записи с индикатором громкости;
- обновляемый черновик распознавания без наложения фрагментов;
- финальная переработка всей записи перед единственной вставкой текста;
- режим обычного общения и режим подготовки промпта для AI;
- профили ChatGPT, Claude, Gemini и универсальный профиль;
- необязательная локальная обработка через Ollama и Qwen3;
- ускоренный CPU-режим Whisper Small;
- отключённые по умолчанию звуки начала и остановки;
- встроенная проверка и безопасная установка обновлений через GitHub Releases.
